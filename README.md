# 基于行为树的魂类游戏智能敌人AI系统

这个项目展示了如何使用行为树来管理敌人AI，以实现像《黑暗之魂》风格的战斗行为。

## 技术栈
- Unity 2022.3 LTS
- Behavior Designer插件（用于行为树实现）
- C#编程语言

## 功能概述
- 敌人AI具备巡逻、追击、攻击、闪避、格挡等多种行为。
- 基于玩家的位置、血量等动态变化的行为模式。
- 支持AI的阶段转换，当敌人血量低于50%时进入二阶段，攻击模式和频率发生变化。

## 安装与运行
1. 克隆此仓库：
   ```bash
   git clone <https://github.com/johnsmithmichael/Souls-like-EnemyAI>
   ```
2. 打开Unity并导入项目。
3. 按照项目中的说明运行场景并与敌人互动。

## 项目目标
本项目旨在展示使用行为树设计和实现复杂AI决策的能力，适合作为游戏AI系统的求职demo。
